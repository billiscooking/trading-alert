# iOS 捷徑設定教學 — 交易緊急警報

> 在 iPhone 桌面放一個按鈕，按下去就打電話叫醒所有人。

---

## 你需要的資訊

請找管理員（Leo）取得以下資訊，當面設定或私訊傳送：
- Twilio Account SID
- Twilio Auth Token
- Twilio 電話號碼：`+16623761837`
- 要撥打的對象號碼（格式：`+886912345678`）

## 捷徑 A：一鍵緊急警報（預設訊息）

### 建立步驟

1. 打開 iPhone「捷徑」App
2. 點右上角 `+` 建立新捷徑
3. 點上方名稱，改名為「緊急警報」
4. 依序加入以下動作：

#### 動作 1：文字
- 搜尋「文字」→ 選「文字」
- 輸入以下內容（整段貼上）：
```
<Response><Say language="zh-TW" loop="3">&#32202;&#24613;&#35686;&#22577;&#65281;&#24066;&#22580;&#20986;&#29694;&#37325;&#22823;&#20107;&#20214;&#65292;&#35531;&#31435;&#21363;&#26597;&#30475;&#12290;Emergency alert! Check the market now.</Say><Pause length="2"/><Say language="en-US">If you received this alert, please hang up.</Say></Response>
```

#### 動作 2：取得 URL 內容（第 1 個號碼）
- 搜尋「取得 URL 內容」→ 加入
- URL 填入：
```
https://api.twilio.com/2010-04-01/Accounts/你的ACCOUNT_SID/Calls.json
```
- 方法改為：`POST`
- 點「加入新標頭」：
  - 鍵：`Authorization`
  - 值：`Basic base64值`（見下方「產生 Base64」教學）
- 點「加入新表單」，依序加入三個欄位：
  - `To`：`+886912345678`（對方號碼）
  - `From`：`+16623761837`
  - `Twiml`：選上面的「文字」變數

#### 如果要打給多人
- 重複「動作 2」，只改 `To` 的號碼

#### 動作 3：顯示提示
- 搜尋「顯示提示」→ 加入
- 輸入：`警報已發出！`

### 加到桌面
1. 點捷徑右上角 `⋯` → 「加入主畫面」
2. 選一個醒目的圖示和顏色（建議紅色 + 警鈴圖示）

## 捷徑 B：自訂訊息警報

跟捷徑 A 相同，只多一個步驟：

### 在最前面多加一個動作：要求輸入
- 搜尋「要求輸入」→ 加入
- 提示輸入：`輸入警報訊息`
- 預設答案留空

### 修改動作 1（文字）
把固定訊息改成：
```
<Response><Say language="zh-TW" loop="3">提供的輸入</Say><Pause length="2"/><Say language="en-US">If you received this alert, please hang up.</Say></Response>
```
注意：`提供的輸入` 要選捷徑裡的變數（藍色泡泡），不是打字。

## 產生 Base64 認證值

Twilio API 需要 `Account SID:Auth Token` 的 Base64 編碼。

### 方法：用 iPhone 捷徑產生
1. 另建一個捷徑，命名「產生 Base64」
2. 加入動作「文字」→ 輸入 `你的AccountSID:你的AuthToken`
3. 加入動作「Base64 編碼」→ 選「編碼」
4. 加入動作「拷貝到剪貼簿」
5. 執行一次 → 剪貼簿裡就是 Base64 值
6. 回到警報捷徑，Authorization 的值填：`Basic 貼上的Base64值`

> 產生完可以刪掉這個輔助捷徑。

## 手機端設定（接收方）

> **重要發現**：iOS 靜音模式會覆蓋一切，即使白名單來電也沒聲音。
> 必須用「勿擾模式」取代「靜音模式」，才能讓警報來電有聲音。

### Step 1：勿擾白名單設定
1. 把 `+16623761837` 存為聯絡人（名稱例如「交易警報」）
2. 設定 → 專注模式 → 勿擾模式 → 允許通知 → 人員 → 加入「交易警報」
3. 效果：勿擾模式下，交易警報來電會正常響鈴

### Step 2：靜音 / 勿擾自動聯動（iPhone 15 Pro 以上）

日常習慣不用改！用 iOS 自動化讓靜音和勿擾自動互斥：

#### 自動化 1：勿擾開啟 → 靜音自動關閉
1. 捷徑 App → 自動化 → `+`
2. 何時：「勿擾模式」→ 選「開啟時」
3. 執行：搜尋「設定靜音模式」→ 選「關閉」
4. 設為「立即執行」

#### 自動化 2：勿擾關閉 → 靜音自動開啟
1. 捷徑 App → 自動化 → `+`
2. 何時：「勿擾模式」→ 選「關閉時」
3. 執行：搜尋「設定靜音模式」→ 選「開啟」
4. 設為「立即執行」

> 設好後的效果：
> - 開勿擾 → 靜音自動關掉 → 警報來電有聲音
> - 關勿擾 → 靜音自動開回來 → 日常保持靜音
> - 你只需要像以前一樣切勿擾，靜音會自動跟著切換

### 備用方案：重複來電穿透
即使沒做以上設定，同一號碼 3 分鐘內打兩次，iOS 勿擾模式會自動放行第二通。

## 測試

設定完成後，先打一通給自己測試：
- `To` 填自己的號碼
- 確認手機狀態：**靜音 OFF + 勿擾 ON**
- 按下捷徑，確認有收到電話且聽到語音

確認沒問題後，再改回正式的對象號碼。
