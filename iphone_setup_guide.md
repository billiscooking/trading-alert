# iPhone 端設定指南 — 交易警報系統

> 讓警報來電能穿透靜音叫醒你，同時保持日常使用體驗不變。
> 本指南寫給非技術人員，照著做就好。

---

## 為什麼需要改設定？

交易警報系統會用電話叫醒你。但 iPhone 的「靜音模式」會擋掉所有來電聲音，包括警報電話。

**解法**：用「勿擾模式」取代「靜音模式」。效果幾乎一樣（通知靜音），但可以設白名單讓警報電話響。

改完之後你的日常體驗不會有變化，但需要做以下設定來避免副作用。

---

## 設定步驟（共 7 步，約 10 分鐘）

### Step 1：儲存警報號碼為聯絡人
1. 打開「電話」App → 聯絡人 → 右上角 `+`
2. 名稱：`交易警報`
3. 電話號碼：`+16623761837`
4. 儲存

### Step 2：把警報號碼加入勿擾白名單
1. 設定 → 專注模式 → 勿擾模式
2. 「允許通知」→ 人員 → 加入「交易警報」
3. **效果**：開勿擾時，只有這個號碼的來電會正常響鈴

### Step 3：允許所有來電響鈴
> 如果不做這步，開勿擾後所有來電都不會響，你會漏接電話。

1. 設定 → 專注模式 → 勿擾模式 → 允許通知 → 人員
2. 「允許來電」→ 選「**所有人**」
3. **效果**：勿擾只靜音 app 通知（Line、Telegram 等），所有來電正常響鈴

### Step 4：讓通知在用手機時正常彈出（懸浮視窗）
> 如果不做這步，開勿擾後所有通知都會消失在通知中心，你在用手機時也看不到。

1. 設定 → 專注模式 → 勿擾模式 → 通知選項
2. 「將通知設為靜音」改成「**鎖定中**」（不要選「永遠」）
3. **效果**：
   - 用手機時 → 通知正常彈出懸浮視窗（沒聲音）
   - 螢幕鎖定時 → 通知完全靜音

### Step 5：靜音 / 勿擾自動聯動
> 如果不做這步，你每次切勿擾都要手動再切靜音，很麻煩。

在「捷徑」App 建立兩個自動化：

#### 自動化 A：開勿擾 → 自動關靜音
1. 捷徑 App → 自動化 → 右上角 `+`
2. 何時：勿擾模式 →「開啟時」
3. 執行：搜尋「設定靜音模式」→ 選「關閉」
4. 設為「立即執行」

#### 自動化 B：關勿擾 → 自動開靜音
1. 捷徑 App → 自動化 → 右上角 `+`
2. 何時：勿擾模式 →「關閉時」
3. 執行：搜尋「設定靜音模式」→ 選「開啟」
4. 設為「立即執行」

> **效果**：你只需要像以前一樣切勿擾，靜音會自動反向切換。
> - 開勿擾 = 靜音自動關 → 警報來電有聲音
> - 關勿擾 = 靜音自動開 → 回到日常靜音

**注意**：iPhone 15 Pro 以上才能用捷徑控制靜音。如果搜不到「設定靜音模式」，需先到捷徑 App → 資料庫，自訂靜音模式開關按鈕。

### Step 6：關閉個別 App 的通知音效
> 如果不做這步，靜音關掉後，Line、Telegram 等 app 傳訊息會發出聲音。

到各 app 內部關閉音效：
- **Line**：Line → 設定 → 通知 → 關閉「聲音」
- **Telegram**：Telegram → 設定 → 通知和音效 → 關閉音效
- 其他會發出聲音的 app 比照辦理

### Step 7：建立警報捷徑（選做，給觸發者用）
> 這步是給「需要按按鈕觸發警報的人」做的。如果你只是接收方，可以跳過。

詳見 `shortcut_setup_guide.md`。

## 設定前後對比

### 改之前（靜音模式）
| 場景 | 行為 |
|------|------|
| 日常使用 | 靜音 ON，所有通知沒聲音 |
| 收到訊息 | 沒聲音，看螢幕才知道 |
| 交易警報來電 | **沒聲音，可能漏接** |
| 睡覺時警報 | **完全聽不到** |

### 改之後（勿擾模式 + 白名單）
| 場景 | 行為 |
|------|------|
| 日常使用（勿擾 ON） | 所有通知沒聲音，但會彈出懸浮視窗 |
| 收到訊息 | 沒聲音，看螢幕就能看到（跟以前一樣） |
| 交易警報來電 | **有聲音，正常響鈴** |
| 睡覺時警報 | **會被叫醒** |
| 關掉勿擾 | 靜音自動開，回到完全靜音 |

## 已知副作用與解法

| 副作用 | 原因 | 解法 |
|--------|------|------|
| 漏接一般來電 | 勿擾模式預設會靜音所有來電 | Step 3：允許來電設為「所有人」 |
| Line / Telegram 傳訊息有聲音 | 靜音關掉後 app 預設會發聲 | Step 6：到 app 內部關閉音效 |
| 拍照 / 截圖有快門聲 | 靜音 OFF 時 iOS 會播放快門聲 | 無解，iOS 限制。拍照前手動開靜音，或接受有聲音 |
| 通知不彈出懸浮視窗 | 勿擾設定中「將通知設為靜音」設為「永遠」 | Step 4：改成「鎖定中」 |
| 切勿擾時還要手動切靜音 | 兩者獨立運作 | Step 5：設定自動聯動 |

---

## 實測結論

- 靜音 ON → 所有來電無聲音，白名單也無效（沒有任何例外）
- 勿擾 ON + 靜音 OFF + 白名單 → 來電有聲音（唯一穩定方案）
- 能穿透靜音開關的只有：鬧鐘、計時器、政府緊急警報
- iOS 的靜音是硬體級覆蓋，無法用軟體繞過
